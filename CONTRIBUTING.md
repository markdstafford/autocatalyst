# Contributing

Autocatalyst is in early development and is **not accepting external contributions yet**. The codebase
is being built from a settled set of concept docs and architecture decisions, and the contribution
process, coding standards, and license are not yet in place for outside work.

If you are interested, check back later. This file will describe how to contribute once the project is
ready.

## Understanding the system

The documentation corpus is the way in. Start with `AGENTS.md` (the map), then the central overview
[`context-human/spec.md`](context-human/spec.md), the [concept index](context-human/concepts/index.md),
and the [ADR index](context-human/adrs/index.md).
